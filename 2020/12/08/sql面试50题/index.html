sql面试50题 来自猴子数据分析

首先表的关系明确下来 ：

简单查询部分

1.查询姓猴的学生名单、最后一个字带猴的、查询姓名中带猴的

解决思路：明显是使用通配符筛选即可 通配符有% _ []

代码：

SELECT * FROM student WHERE 姓名 like '猴%';

SELECT * FROM student WHERE 姓名 like '%猴';

SELECT * FROM student WHERE 姓名 like '%猴%';

2.查询姓“孟”老师的个数

解决思路 ：查询个数用count() 姓名继续是通配符匹配

代码：

SELECT count(教师号) FROM teacher where 教师姓名 like '孟%';

汇总分析

3查询课程号为”0002”的总成绩

思路：很简单筛选出0002的课程然后使用聚合函数sum()

代码：

SELECT SUM(成绩) FROM score where 课程号='0002';

4查询选了课程的学生人数

思路：只要在成绩表中有记录说明就选了课

代码：SELECT COUNT(DISTINCT 学号) FROM score;

5分组：查询各科成绩最高和最低

思路：显然只要进行分组之后返回最大值即可

select 课程号,max(成绩) as 最高分,min(成绩) as 最低分
from score
group by 课程号;

6：查询每门课程被选修的学生数

思路：很简单分组完后统计学号个数即可

SELECT a.课程名称,b.课程号,COUNT(学号) FROM course a INNER JOIN score b ON a.课程号=b.课程号 GROUP BY 课程号;

7：查询男生女生人数

思路：仍然是分组之后统计

代码:

select 性别,count(*)
from student
group by 性别;

8：查询平均成绩大于60分学生的学号和平均成绩

思路 要查的是学号和平均成绩 然后条件是分数大于60分依据学号分组

代码

SELECT 学号,AVG(成绩) FROM score GROUP BY 学号 HAVING AVG(成绩)>60;

9 查询至少选修两门课程的学生学号

思路：仍然是分组统计 选出课程数大于二的即可

代码：

SELECT 学号,COUNT(课程号) as 选修课程数目 FROM score GROUP BY 学号 HAVING COUNT(课程号);

10 查询同名同姓 学生名单并统计同名人数

思路 这个稍微绕了一下，如何判断姓名相同，答案就是分组统计 分组后人数大于二等于2的就算作同名姓

代码: SELECT 姓名,COUNT(*) as num FROM student GROUP BY 姓名 HAVING COUNT(*)>=2;

11 查询不及格的课程并按课程号从大到小排列

思路 ：清晰明了 排个序即可

select 课程号
from score
where 成绩<60
order by 课程号 desc;

12 查询每门课程的平均成绩，结果按平均成绩升序排序，平均成绩相同时，按课程号降序排列

思路：分组 聚合算出平均成绩 然后排序即可

代码:SELECT 课程号,AVG(成绩) as 平均成绩 FROM score GROUP BY 课程号 ORDER BY 平均成绩,课程号 DESC;

13 检索课程编号为“0004”且分数小于60的学生学号，结果按按分数降序排列

思路：就是筛选排序

select 学号
from score
where 课程号='04' and 成绩 <60
order by 成绩 desc;

14 统计每门课程的学生选修人数(超过2人的课程才统计)

要求输出课程号和选修人数，查询结果按人数降序排序，若人数相同，按课程号升序排序

思路 输出的是课程号和人数

代码：select 课程号, count(学号) as '选修人数'
from score
group by 课程号
having count(学号)>2
order by count(学号) desc,课程号 asc;

15 查询两门以上不及格课程的同学的学号及其平均成绩

思路：需要输出的是学号以及平均成绩 筛选条件是两门以上课程不及格

select 学号, avg(成绩) as 平均成绩
from score
where 成绩 <60
group by 学号
having count(课程号)>2;

16 查询学生的总成绩并进行排名

思路：简单的加总

select 学号 ,sum(成绩) from score
group by 学号
order by sum(成绩);

17 查询平均成绩大于60分的学生的学号和平均成绩

思路：分组聚集

代码：

SELECT 学号,AVG(成绩) FROM score GROUP BY 学号 HAVING AVG(成绩)>60;

复杂查询

18 查询所有课程成绩小于60分学生的学号、姓名

思路：1 使用子查询 2.使用多表查询

代码：

SELECT a.学号,b.姓名 FROM score a,student b WHERE a.学号=b.学号 and a.成绩<60

或者

select 学号,姓名 from student where 学号 in ( select 学号
from student
where 成绩 < 60
);

19 查询没有学全所有课的学生的学号、姓名

思路：该学生选修的课程数 < 总的课程数

select 学号,姓名from student
where 学号 in(
select 学号
from score
group by 学号
having count(课程号) < (select count(课程号) from course)
);

20 查询出只选修了两门课程的全部学生的学号和姓名

思路：也是很简单的一个子查询即可

select 学号,姓名 from student where 学号 in( select 学号
from score
group by 学号
having count(课程号)=2
);

21 查询1990年出生的学生名单

补充日期函数

思路 筛选年份为1990即可

代码

select 学号,姓名
from student
where year(出生日期)=1990;

22 查询各科成绩前两名的记录

补充知识点：分组取每组最大值、最小值、每组最大的N条记录(top N)

成绩表 ：学号、课程号、成绩

可以使用分组（group by）和汇总函数得到每个组里的一个值（最大值，最小值，平均值等）。但是无法得到成绩最大值所在行的数据，如下

select 课程号,max(成绩) as 最大成绩from score group by 课程号;

看不到学生学号，因为select 指定的列必须要出现在group by中，除非是聚集函数

可以用关联子查询来解决这个问题

select * from score as a where 成绩 = (select max(成绩) from score as b where b.课程号 = a.课程号);

同理分组取最小值也是一样的操作

取每组最大的N条记录

思路：首先查询有哪些分组，然后进行排序限制某几个，最后union 连接在一起

select 课程号,max(成绩) as 最大成绩from score group by 课程号;

(select * from score where 课程号 = '0001' order by 成绩 desc limit 2)

union all

(select * from score where 课程号 = '0002' order by 成绩 desc limit 2)

union all

(select * from score where 课程号 = '0003' order by 成绩 desc limit 2);

23 查询各学生的年龄（精确到月份）

使用时间差函数 timestampdiff

select 学号 ,timestampdiff(month ,出生日期 ,now())/12
from student ;

多表查询

25 查询所有学生的学号、姓名、选课数、总成绩

思路：学号 姓名在学生表中 选课数在成绩表中 选课数是聚合函数统计

select a.学号,a.姓名,count(b.课程号) as 选课数,sum(b.成绩) as 总成绩
from student as a left join score as b
on a.学号 = b.学号
group by a.学号;

注意是left join 因为有的学生是没有选课的

26 查询平均成绩大于85的所有学生的学号、姓名和平均成绩

思路：和上面一题差不多只是多了个筛选条件

SELECT a.学号,a.姓名,AVG(成绩) as 平均成绩 from student a LEFT JOIN score b on a.学号=b.学号 GROUP BY 学号 HAVING AVG(成绩)>85;

27 查询学生的选课情况：学号，姓名，课程号，课程名称

思路 ：分别在三个表中 ，思考三个表如何连接

select a.学号, a.姓名, c.课程号,c.课程名称
from student a inner join score b on a.学号=b.学号
inner join course c on b.课程号=c.课程号;

28 查询出每门课程的及格人数和不及格人数

select 课程号,
sum(case when 成绩>=60 then 1
else 0
end) as 及格人数,
sum(case when 成绩 < 60 then 1
else 0
end) as 不及格人数
from score
group by 课程号;

29 使用分段[100-85],[85-70],[70-60],[<60]来统计各科成绩，分别统计：各分数段人数，课程号和课程名称

思路 和上面代码一致 只不过需要联结一下课程号和课程名

SELECT a.课程号,a.课程名称,
SUM(CASE
WHEN 成绩 between 85 and 100 THEN
1
ELSE
0
END
) as '[85,100]',
SUM(CASE
WHEN 成绩 between 70 and 85 THEN
1
ELSE
0
END
) as '[70,85]',
SUM(CASE
WHEN 成绩 between 60 and 70 THEN
1
ELSE
0
END
) as '[60,70]',
SUM(CASE
WHEN 成绩 <60 THEN
1
ELSE
0
END
) as '不及格'
FROM course a RIGHT JOIN score b
on a.课程号 =b.课程号
group by a.课程号,a.课程名称;

30 查询课程编号为0003且课程成绩在80分以上的学生的学号和姓名

select a.学号,a.姓名
from student as a inner join score as b on a.学号=b.学号
where b.课程号=’0003’ and b.成绩>80;

下面的表

转化为

第一步 使用常量列输出目标表的结构

select 学号,’课程号0001’,’课程号0002’,’课程号0003’ from score;

第二步 使用case when 将内部的数据换成成绩

select 学号,

(case 课程号 when '0001' then 成绩 else 0 end) as '课程号0001',

(case 课程号 when '0002' then 成绩 else 0 end) as '课程号0002',

(case 课程号 when '0003' then 成绩 else 0 end) as '课程号0003'

from score;

结果如下

思考这里是如何实现的 首先第一个case when 对课程号这一列遍历 课程号是001的就选出成绩得到 80 0 0 0 0 48 0 0 这一列 然后 再执行第二列 得到0 90 0 59 0 0 80 0这一列

最后一步 如下进行分组 只要取每个方块里的最大值即可

select 学号,

max(case 课程号 when '0001' then 成绩 else 0 end) as '课程号0001',

max(case 课程号 when '0002' then 成绩 else 0 end) as '课程号0002',

max(case 课程号 when '0003' then 成绩 else 0 end) as '课程号0003'

from scoregroup by 学号;

多表联结的情况

32 检索”0001”课程分数小于60，按分数降序排列的学生信息

思路 很简单只要联结两个表即可

SELECT a.*,b.成绩 FROM student a INNER JOIN score b on a.学号=b.学号 WHERE b.课程号=001 AND b.成绩<60 ORDER BY b.成绩 desc;

33 查询不同老师所教不同课程平均分从高到低显示

思路如下：也是简单的联结完表之后 然后降序排序 主要就是一个多表连接 加上一个排序

select a.教师号,a.教师姓名,avg(c.成绩)
from teacher as a
inner join course as b
on a.教师号= b.教师号
inner join score c on b.课程号= c.课程号
group by a.教师姓名
order by avg(c.成绩) desc;

34 查询课程名称为”数学”，且分数低于60的学生姓名和分数

思路 依然是多表连接加一个筛选 很简单的东西

select a.课程名称,c.姓名,b.成绩

FROM course a INNER JOIN score b on a.课程号=b.课程号

INNER JOIN student c on b.学号=c.学号

WHERE b.成绩<60 AND a.课程名称='数学';

35 查询任何一门课程成绩在70分以上的姓名、课程名称和分数

思路：和上题一样 只要修改筛选条件即可

select a.课程名称,c.姓名,b.成绩`

FROM course a INNER JOIN score b on a.课程号=b.课程号

INNER JOIN student c on b.学号=c.学号

WHERE b.成绩<60 AND a.课程名称='数学';

36 -查询两门及其以上不及格课程的同学的学号，姓名及其平均成绩

思路 这里稍微难一点的部分在于怎么筛选出两门课程及其以上课程不及格 ，可以先筛选不及格的然后再统计

select b.姓名,avg(a.成绩),a.学号 from score as a
inner join student as b
on a.学号 =b.学号
where a.成绩 <60
group by a.学号
having count(a.学号 ) >=2;

37 查询不同课程成绩相同的学生的学生编号、课程编号、学生成绩

思路 学生号 课程号 成绩 都在一个表中 但是要查出 成绩相同课程不同 只要两个score表就可以完成

select distinct a.学号 ,a.成绩 ,a.课程号
from score as a
inner join score as b
on a.学号 =b.学号
where a.成绩 =b.成绩 and a.课程号 != b.课程号;

38 查询课程编号为“0001”的课程比“0002”的课程成绩高的所有学生的学号

思路 需要查询的是学号，条件是 0001 课程成绩比 0002 课程成绩高 这个有点难度 需要在筛选部分考虑好 ，借鉴下别人的写法

select a.学号
from
(select 学号,成绩 from score where 课程号='0001') as a
inner join
(select 学号,成绩 from score where 课程号='0002') as b
on a.学号 =b.学号 where a.成绩 > b.成绩 ;

39 -查询学过“孟扎扎”老师所教的所有课的同学的学号、姓名(很多的表进行联结)

思路：最终要查询的是同学的学号和姓名，根据表的结构 首先在teacher表找孟扎扎的教师号，然后根据这个教师号从course表中找到该教师所教课程号，再从根据这个课程号从score表中找到学号，最后再到student表中找学生姓名

如何去联结这么多表呢？方法是先找到主表 比如这里的最后结果的表学生表，然后再一一联结

SELECT s.学号,s.姓名 FROM student s INNER JOIN score a on s.学号=a.学号

INNER JOIN course b on a.课程号=b.课程号

INNER JOIN teacher c on b.教师号=c.教师号

WHERE c.教师姓名=’孟扎扎’;
